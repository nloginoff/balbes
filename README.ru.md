# Balbes — Твоя AI-команда, живущая в Telegram

> *А что, если твой личный ИИ-помощник умеет писать код, искать в интернете, помнить всё важное и сам пишет тебе — прямо в Telegram?*

**Balbes** — это self-hosted мульти-агентная AI-система, построенная вокруг Telegram-бота. Вдохновлена философией [OpenClaw](https://github.com/openclaw): специализированные агенты вместо одного «всемогущего» ИИ, каждый со своей памятью, инструментами и характером.

Агент **Coder** — это твой личный [Cursor AI](https://cursor.sh) прямо в Telegram. Он читает твой код, пишет новый, запускает команды, делает коммиты и итерируется — ты описываешь задачу, он делает.

**Версия**: 0.4.0 | **Статус**: 🟢 Работает в продакшне | **Лицензия**: MIT

---

## Почему Balbes?

Большинство ИИ-ассистентов — это просто чат-окно без памяти. Balbes — другой:

- **Помнит** — семантическая долгосрочная память в Qdrant + история 7 дней в Redis
- **Действует** — не только отвечает, но и выполняет команды, читает/пишет файлы, ищет в интернете
- **Делегирует** — Оркестратор передаёт задачи агенту Coder в фоновый режим
- **Сам пишет** — система Heartbeat делает агента проактивным: он сам шлёт тебе обновления, идеи или напоминания
- **Твой сервер** — твои данные, твои модели, твои правила

## Главные фичи

### 🤖 Агент Coder — Cursor AI внутри Telegram

Не нужно переключаться между приложениями. Coder — это полноценный coding-ассистент:

- Читает любой файл в проекте (`file_read`)
- Пишет и изменяет файлы (`file_write`)
- Запускает `git`, `grep`, `rg`, `diff`, `tree`, shell-скрипты
- Автоматически делает git-коммиты
- Работает **в фоне** — ты занимаешься своим делом, он работает; когда закончит — пришлёт результат в Telegram

Дай ему задачу типа *«отрефактори скилл поиска под новый провайдер»* — и иди пить кофе.

### 🧠 Долгосрочная память

Агент помнит то, чему ты его учил. Семантический поиск по твоим заметкам, решениям и контексту через Qdrant. `/remember` — сохранить, `/recall` — найти.

### 🔍 Поиск в интернете

Мультипровайдерный поиск с автоматическим fallback:
- **Tavily** — AI-оптимизированные результаты
- **Yandex Search API v2** — русскоязычный поиск через Yandex Cloud
- **Brave** — приватный поиск

Переключение прямо из чата: *«поищи через yandex: ...»*

### 🎙️ Голосовые сообщения

Отправь войсовое — получи текстовый ответ. Транскрипция через `faster-whisper` с опциональной LLM-коррекцией.

### 💓 Heartbeat — агент, который сам пишет

Агент не ждёт, пока его спросят. На основе твоего файла `HEARTBEAT.md` он проактивно шлёт сообщения — обновления по проекту, напоминания, интересные находки. Работает на бесплатной модели для экономии.

### 🔧 Полное управление через Telegram

Никакого Web UI не нужно. Переключение моделей, управление чатами, просмотр логов, управление агентами — всё в одном Telegram-чате.

> **Статус Web UI**: Веб-фронтенд находится в активной разработке, но честно говоря, разработчик не видит в нём особой нужды. Через Telegram-бота всё работает удобно и быстро — это и есть ключевая идея проекта. Web-интерфейс появится, когда появится.

## Возможности

| Фича | Описание |
|------|----------|
| Мульти-агентная оркестрация | Оркестратор делегирует задачи Coder-у в фоне или синхронно |
| Своя модель на каждый чат | Каждый чат может использовать разный LLM (бесплатный → премиум) |
| Мониторинг фоновых задач | Live debug-трейс стримится в Telegram по мере работы агента |
| Версионирование воркспейса | Файлы агента автоматически коммитятся в приватный git-репо |
| Переключение режимов | `/mode ask` (безопасный) / `/mode agent` (полные права разработчика) |
| Голосовые сообщения | `faster-whisper` + опциональная LLM-коррекция |
| Лимиты токенов | Суточные/часовые бюджеты на агента с автоматическим fallback |
| Несколько чатов | Отдельные истории, агенты и настройки на каждый чат |

## Команды Telegram

| Команда | Описание |
|---------|----------|
| `/start` | Начать работу |
| `/help` | Помощь |
| `/agents` | Список агентов / сменить агента |
| `/chats` | Список чатов / сменить чат |
| `/newchat` | Создать новый чат |
| `/rename` | Переименовать текущий чат |
| `/model` | Выбрать модель (free / cheap / medium / premium) |
| `/clear` | Очистить историю чата |
| `/remember` | Сохранить в долгосрочную память (Qdrant) |
| `/recall` | Найти в долгосрочной памяти |
| `/heartbeat` | Запустить heartbeat прямо сейчас |
| `/debug` | Включить/выключить live debug-трейс в чате |
| `/mode` | Переключить режим: `ask` / `agent` |
| `/tasks` | Реестр задач: текущие и завершённые |
| `/stop` | Остановить текущее действие агента |
| `/status` | Статус системы |

### Фоновые задачи

```
Ты → Оркестратор → delegate_to_agent(coder, background=true)
                          │
                   Coder работает в фоне...
                          │ (каждые 5 сек)
                   bg_monitor_loop ← polls /api/v1/tasks/bg/events
                          │
                   Telegram: live трейс + финальный результат ✅
```

## Технологический стек

| Слой | Технология |
|------|-----------|
| Язык | Python 3.13 |
| Фреймворк | FastAPI + asyncio (кастомный agent loop) |
| LLM-шлюз | [OpenRouter](https://openrouter.ai) (все популярные модели) |
| Telegram | python-telegram-bot v22+ |
| Быстрая память | Redis (история чатов, сессии, флаги) |
| Долгосрочная память | Qdrant (семантический векторный поиск) |
| База данных | PostgreSQL |
| Транскрипция голоса | faster-whisper + ffmpeg |
| Поиск | Tavily · Yandex Search API v2 · Brave |
| Инфраструктура | Docker Compose |
| Качество кода | ruff · pre-commit |

## Быстрый старт

### Что нужно

- Linux-сервер (VPS или локальный)
- Docker + Docker Compose
- Python 3.13
- Telegram bot token ([@BotFather](https://t.me/botfather))
- API-ключ [OpenRouter](https://openrouter.ai)

### 1. Клонировать и настроить

```bash
git clone https://github.com/your-username/balbes.git
cd balbes
cp .env.example .env.prod
```

Заполни `.env.prod`:

```bash
OPENROUTER_API_KEY=sk-or-v1-...
TELEGRAM_BOT_TOKEN=...
TELEGRAM_USER_ID=ТВОЙ_TELEGRAM_ID  # узнай через @userinfobot
WEB_AUTH_TOKEN=любой-случайный-секрет
JWT_SECRET=любой-случайный-секрет
POSTGRES_PASSWORD=твой-пароль
```

Опционально (для веб-поиска):

```bash
TAVILY_API_KEY=tvly-...            # https://tavily.com
YANDEX_SEARCH_KEY=AQVN...         # API-ключ Yandex Cloud
YANDEX_FOLDER_ID=b1g...           # ID каталога Yandex Cloud
BRAVE_SEARCH_KEY=...               # https://api.search.brave.com
```

### 2. Запустить инфраструктуру

```bash
docker compose -f docker-compose.prod.yml up -d
```

### 3. Запустить сервисы

```bash
bash scripts/start_prod.sh
```

### 4. Написать `/start` в Telegram

## Конфигурация

Поведение агентов контролируется через:

- **`config/providers.yaml`** — модели, тиры, настройки агентов, конфиг скиллов, heartbeat
- **`data/agents/{agent}/config.yaml`** — переопределения на уровне агента (высший приоритет)
- **`data/agents/{agent}/AGENTS.md`** — инструкции поведения, которые видит LLM
- **`data/agents/{agent}/MEMORY.md`** — постоянный контекст, который агент всегда несёт с собой
- **`.env.prod`** — секреты и параметры инфраструктуры

Полный справочник: [`docs/ru/CONFIGURATION.md`](docs/ru/CONFIGURATION.md)
